# Knowledge Engine

Short description: A small knowledge/search engine for ticket solutions.

## Structure
- backend/
- frontend/
- requirements.txt

## Run (Windows PowerShell)
1. python -m venv venv
2. .\venv\Scripts\Activate.ps1
3. pip install -r requirements.txt
4. python backend/app.py

**Do NOT commit .env, .db, .pkl or the venv folder.**
